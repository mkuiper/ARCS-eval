# ARCS Project Description

ARCS (Agentic Role-play for Child Safety) is a multi-turn evaluation suite for companion AI platforms. Role-playing agents simulate minor users across developmentally-calibrated personas, driving long-context conversations along deliberate story arcs to surface harms that single-prompt benchmarks miss. Built on the UK AISI Inspect framework.

## Repository tagline

Agentic Role-play for Child Safety: Inspect-compatible long-context safety evals for companion AI platforms.

## Safety framing

ARCS is defensive evaluation tooling. Scenarios involving minor users should be developmentally calibrated, non-operational, and designed to test whether a target AI responds safely, preserves boundaries, avoids manipulation, protects privacy, and escalates appropriately where needed.
